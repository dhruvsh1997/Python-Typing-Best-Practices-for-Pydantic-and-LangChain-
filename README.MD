## ğŸ“– Learning Modules

### Module 1: ğŸ”¤ Basic Types


### Module 2: ğŸ”€ Union & Literal Types
Handle multiple possible types and restrict values to specific options.

```python

# Restrict to specific values

```

### Module 3: ğŸ“‹ TypedDict & Callable
Structure dictionaries without classes and type function parameters.

```python

    name: str
    email: str
    age: int

# Function type specification

```

### Module 4: ğŸ§¬ Generic Programming
Create reusable, type-safe containers and functions.

```python

class Container(Generic[T]):
    def __init__(self, item: T):
        self.item = item
    
    def get_item(self) -> T:
        return self.item

# Usage

str_container = Container[str]("Hello")
```

### ğŸ› ï¸ LangChain Tool with Typing

```python
from langchain_core.tools import tool
from pydantic import BaseModel, Field

class CalculatorInput(BaseModel):
    a: float = Field(description="First number")
    b: float = Field(description="Second number")
    operation: Literal["+", "-", "*", "/"] = Field(description="Operation")

@tool
def calculator(a: float, b: float, operation: str) -> float:
    """Perform basic arithmetic operations."""
    operations = {"+": a + b, "-": a - b, "*": a * b, "/": a / b}
    return operations[operation]
```

## ğŸŒŸ Advanced Features

### ğŸ§ª Generic Container Classes

```python
from typing import TypeVar, Generic, Protocol

T = TypeVar('T')
K = TypeVar('K')
V = TypeVar('V')

class Repository(Generic[T]):
    """Generic repository pattern."""
    
    def __init__(self):
        self._items: List[T] = []
    
    def add(self, item: T) -> None:
        self._items.append(item)
    
    def get_all(self) -> List[T]:
        return self._items.copy()

# Usage
user_repo = Repository[User]()
product_repo = Repository[Product]()
```

## ğŸ”— Integration Examples

### ğŸ¤– Complete Agent Example

```python
from langgraph.graph import StateGraph, END
from langchain_core.tools import tool
from typing import TypedDict, Annotated, List, Literal

# State definition
class AgentState(TypedDict):
    messages: Annotated[List[dict], add_messages]
    current_step: Literal["input", "processing", "output"]
    result: Optional[str]

# Tool definition
@tool
def process_query(query: str) -> str:
    """Process user query and return result."""
    return f"Processed: {query}"

# Node functions
def input_node(state: AgentState) -> AgentState:
    return {"current_step": "processing"}

def process_node(state: AgentState) -> AgentState:
    # Processing logic here
    return {"current_step": "output", "result": "Success"}

# Build graph
graph = StateGraph(AgentState)
graph.add_node("input", input_node)
graph.add_node("process", process_node)
graph.add_edge("input", "process")
graph.add_edge("process", END)
```
